# PerFin OS Project Brief

## Product Summary

PerFin OS is a personal finance operating system for tracking expenses, budgets, savings goals, receipts, mapped spending, reports, and educational planning insights.

## Target Users

- Budget-conscious students and early-career professionals.
- Users who want lightweight expense visibility without bank integrations.
- Future freemium users who may upgrade for AI, receipt, sync, and advanced planning features.

## v1.0.0 Scope

- Login-first app entry after logo animation.
- Local-only guest mode.
- Firebase-authenticated user workspaces.
- Guest-to-account import prompt.
- Expense CRUD with numeric guards.
- Current-location default and place-search selection.
- Colorful heatmap and category pins.
- Multi-image receipt attachment metadata.
- AI Reports + Planner fallback using aggregate-only data.
- Placeholder integration states for missing keys.

## Out Of Scope For v1.0.0

- Payments and subscriptions.
- Bank account linking.
- Real financial advice.
- Firebase Storage for receipts.
- Full production receipt signing implementation.
- Automated end-to-end test suite.
